자바스크립트에서 사용하는 함수들

자바스크립트에서 배열은 사용빈도가 높은 객체이다. 자바스크립트 배열객체는 배열을 다루기 위한 유용한 속성과 함수들이 제공된다.

  • Array.prototype.push() : 마지막요소로 추가 (List, add 랑 비슷)
  • Array.prototype.pop() : 마지막요소 제거 (List, removeLast 랑 비슷)
  • Array.prototype.unshift() : 맨 처음요소에 추가 (List, addFirst 랑 비슷)
  • Array.prototype.shift() : 맨 처음요소 제거 (List, removeFirst 랑 비슷)
  • Array.prototype.splice() : 배열요소 변경 (몇번째위치부터 몇개 제거하고 머로 바꿀껀가)
  • Array.prototype.join() : 배열요소 변경 (몇번째위치부터 몇개 제거하고 머로 바꿀껀가)
